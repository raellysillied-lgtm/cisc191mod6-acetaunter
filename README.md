# Module 6 Lab: Generics, Collections, and Advanced Streams
## 1v1 Game Server Theme

In this lab, you will use a 1v1 matching game server theme to practice:
- generics
- collections
- advanced stream pipelines

## Learning Goals

By the end of this lab, you should be able to:
1. implement a generic collection-backed class
2. use List, Set, Map, and Queue-like behavior appropriately
3. write stream pipelines with filtering, sorting, mapping, grouping, and averaging
4. choose readable collection and stream solutions to game-server style problems

## Files to Focus On
- GenericMatchQueue.java
- GameServerAnalytics.java

## Rules
- Do not rename classes or methods.
- Do not edit the tests.
- Keep code in the package edu.sdccd.cisc191.
- Use streams where the prompt suggests stream processing.
- Throw IllegalStateException when dequeue or peek is called on an empty queue.
